Reference Library

Muse Models

A visual reference library of AI-generated model faces and outfits for use across Purplle's private label brands — Alps Goodness, Good Vibes, Dermdoc, and NY Bae. Pick a face, pick an outfit, and use the references as a starting point for any model shot you generate.

How to use

  1. Scroll down to the gallery below to browse the full library.
  2. Find a model that fits the brand and shot you're working on — each brand has its own approved models, with face close-ups and outfit options for each.
  3. Use the chosen reference image as a prompt input or visual anchor when generating new shots in Higgsfield (Nano Banana Pro / NB2).
  4. Stay consistent — once a model is locked to a brand, use the same face across that brand's shots.

What you'll need

  • Just a browser — the gallery lives on this page
  • Higgsfield access if you're generating new shots from these references
  • Brand context — know which brand you're shooting for
