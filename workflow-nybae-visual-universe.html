Skill

NY Bae Visual Universe

A Claude skill that gives Claude the full NY Bae visual universe: the pink-owns-the-frame palette, the studio-clean lighting, the four layout splits, and the three approved model looks (Fair Traditional, Medium Modern, Dusky Fresh). Use it to write or review image prompts so any NY Bae visual — PDP image, banner, social post, model shot — feels unmistakably on-brand.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and mention NY Bae or any NY Bae product (Velvet Mousse Lipstick, Pro Strobe Cream, kajal, etc.) — the skill triggers automatically.
  3. Tell Claude what you're making — a PDP image, banner, social post, or model shot — and which product it's for.
  4. Claude returns an image prompt that wraps the brief in NY Bae's palette, lighting, an appropriate layout split, and the right approved model look. Pair with the NBP Whisperer skill for the final NBP / NB2 prompt mechanics.
  5. To review an existing image, paste it into Claude and ask if it's on-brand — the skill checks it against the universe rules.
  6. Important: the 3D pink heart motif is post-production only — never include hearts in image-generation prompts.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • Higgsfield account with NBP / NB2 access for generating the images
  • Optionally pair with the NBP Whisperer skill for the final prompt mechanics
