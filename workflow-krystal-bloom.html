Skill

Krystal Bloom

A Claude skill that gives Claude the Krystal Bloom visual universe for Jahn. — the prismatic light-through-crystal signature, the soft pastel dreamscape palette, and the casting and composition rules. Use it to write or review prompts so any Krystal Bloom visual feels premium, calm, feminine, and unmistakably in-world.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and mention Krystal Bloom, Jahn., prismatic, or the fragrance line — the skill triggers automatically.
  3. Tell Claude what you're making — a campaign visual, packaging shot, social post, banner, or mood board.
  4. Claude returns an image prompt grounded in the Krystal Bloom world: prismatic light reflections, soft pastel dreamscape, feminine energy, premium calm.
  5. To review an existing image, paste it into Claude and ask if it's on-brand — the skill checks it against the universe rules.
  6. Important: Malmal is a separate Jahn. universe (for the IndiPop Superstar makeup range). Do not mix the two.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • Higgsfield account with NBP / NB2 access for generating the images
  • Optionally pair with the NBP Whisperer skill for the final prompt mechanics
