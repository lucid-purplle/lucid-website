Skill

A+ from PDP

A Claude skill that turns an existing Purplle private-brand PDP into Amazon A+ / off-platform module prompts for Nano Banana Pro — for Alps Goodness, Good Vibes, Dermdoc, and NY Bae. Nothing is designed from scratch: the PDP is the source of truth for the pack, palette, type, and claims. The skill reads the PDP, describes the proposed module in plain language for sign-off, then writes the full NBP prompt.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Upload all the PDP images for the product, plus the Amazon or Purplle product page link — then ask for an A+: "make an A+ from this PDP", "A+ module", or "listing banner from this PDP" will trigger the skill.
  3. Claude reads the PDP and extracts the exact product name, claims (verbatim), social proof, pack description, palette, and photography style — nothing is invented.
  4. Before any prompt is written, Claude describes the proposed module in plain language — layout zones, headline, supporting text — for your sign-off.
  5. Once approved, Claude writes the full NBP prompt: match-the-reference instructions for brand identity, new-build instructions for layout and text, every line of text verbatim in quotes, and one headline box only.
  6. Set the aspect ratio in Higgsfield yourself and upload the PDP hero as the reference image — the ratio is never written into the prompt. Generate, review, iterate.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • The finished PDP images and the product's Amazon / Purplle link
  • Higgsfield access for the Nano Banana Pro generation
