Skill

Alps Goodness PDP Imagery

A Claude skill that turns an Alps Goodness PDP content doc into per-slide image-generation prompts — hero, texture, ingredient, how-to-use, before/after, how-it-feels, rooted-in-ritual, pair-it-with, what's-inside, and the rest. Each prompt is built around an archetype, references real product and model images via placeholders, and leaves the deliberate negative-space zone clear for post-production text. It writes imagery only — it does not plan the PDP, write its copy, or do layout.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and mention Alps Goodness, AG PDP, or any AG product — the skill triggers automatically. Upload the PDP content doc and the reference images you have ([MODEL] portrait, [HERO PRODUCT] packshot, plus [PAIR PRODUCT] or [KIT] packshots where relevant).
  3. Claude reads the content doc slide by slide and classifies each into an archetype (Hero, Texture / In-Use, Ingredient Spotlight, How to Use, Before/After, How It Feels, Rooted in Ritual, Pair It With, What's Inside). Post-only slides (Reviews, Closing, pure stat) are flagged as no generation.
  4. For each image-led slide, Claude returns the slide number + archetype, the references it expects, and the art-direction prompt — scene, props, palette move, light, composition, and the negative-space zone — written to hand off to the NBP Whisperer skill for final NBP / NB2 structuring.
  5. If the deck has a non-standard slide (a ratio chart, a one-powder-many-rituals multi-use slide), the skill asks you how to execute it rather than guessing the layout.
  6. Pair with the Alps Goodness Visual Universe (Soft Living) skill for palette / lighting / props, and with NBP Whisperer for the final prompt mechanics.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • The PDP content doc and the relevant reference images for the product (model portrait, hero packshot, plus pair-product or kit packshots if those slides exist)
  • Higgsfield account with NBP / NB2 access for actually generating the images
  • Optionally pair with the Alps Goodness Visual Universe (Soft Living) and NBP Whisperer skills
