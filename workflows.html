Workflows

Ready-to-use AI workflows built by Lucid for teams across Purplle.

Skill

NBP Whisperer

A Claude skill that turns Claude into a dedicated prompt writer for Nano Banana Pro and Nano Banana 2 on Higgsfield. Give it a brief, get back a model recommendation and a ready-to-paste prompt.
Skill

Artwork Checker

A Claude skill built for the Hotline (same-day delivery) team that catches errors on packaging artwork before it goes to print. Upload a label, carton, or kit and it flags only the actual issues — FOP, BOP, compliance, copy quality, and manufacturer details.
Prompt Library

PDP Video Prompts

A growing library of ready-to-use image and video prompts for producing PDP videos across brands. Each product has its own sheet with shot-by-shot prompts — hero shot, concern, texture, application, before/after, beauty shot, closing — written for Nano Banana Pro and Seedance 2.0.
Reference Library

Muse Models

A visual reference library of AI-generated model faces and outfits for use across Purplle's private label brands — Alps Goodness, Good Vibes, Dermdoc, and NY Bae. Pick a face, pick an outfit, and use the references as a starting point for any model shot you generate.
Skill

Good Vibes PDP Builder

A Claude skill that conceptualises, structures, and writes copy for Good Vibes PDPs on the Purplle app — slide by slide, in the Good Vibes voice. It plans the slide structure, writes copy, drafts a Nano Banana Pro image prompt per slide, and reviews finished designs.
Skill

NY Bae PDP Builder

A Claude skill that conceptualises, structures, and writes copy for NY Bae PDPs on the Purplle app — slide by slide, in the NY Bae voice. It plans the slide structure, writes copy, drafts a simple image-generation prompt per slide, and reviews finished designs.
Skill

NY Bae Visual Universe

A Claude skill that wraps any image-generation prompt in NY Bae's pink-on-pink, dewy, premium-glossy visual universe — palette, lighting, the four layout splits, and three approved model looks. Pair it with NBP Whisperer to write or review NBP / NB2 prompts for any NY Bae visual.
Skill

Alps Goodness Visual Universe

A Claude skill that wraps any image-generation prompt in Soft Living — Alps Goodness's light, airy, lived-in visual universe, built on the rule that the ritual is already complete (show the finished state, never the process).
Skill

IndiPop Visual Universe

A Claude skill that wraps any image-generation prompt in IndiPop's Malmal universe for the Jahn. Superstar makeup range — the dusty Indian-craft palette, soft pastel studio backdrops, and dreamy atmospheric haze that make every shot read as one coherent campaign.
Skill

Krystal Bloom Visual Universe

A Claude skill that wraps any image-generation prompt in Krystal Bloom — Jahn.'s luminous, prismatic, dreamy-feminine visual universe for the fragrance line. Use it to write or review prompts so every Krystal Bloom visual feels premium, calm, feminine, and unmistakably in-world.

DERMDOC Tone of Voice

Whenever you're writing any copy for Dermdoc, paste this Tone of Voice guide into your chat first. It keeps the brand voice consistent.
