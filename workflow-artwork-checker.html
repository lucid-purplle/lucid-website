Skill

Artwork Checker

How it works

Things that used to slip past three rounds of review, now caught in one.

  • MRP missing the ₹ symbol — easy to miss, lands you in compliance trouble. Caught.
  • Net Quantity mismatch — the FOP says 50g, the BOP says 50ml, the e-com listing says 45g. Caught.
  • "MADE IN INDIA" missing or in the wrong case — required text, exact format. Caught.
  • Manufacturer block incomplete — missing license number, address line, customer care email, or in the wrong order. Caught.
  • Batch fields blank — MFG date, EXP date, Batch No., Best Before placeholders left empty in the print-ready file. Caught.
  • Approved manufacturer not used — artwork lists a manufacturer who isn't on the approved list. Caught.
  • Barcode positioned wrong, too small, or distorted — quietly fails at scanning. Caught.
  • Pantone colours called out but the artwork's in CMYK — a printer's nightmare. Caught.
  • USP text rephrased mid-cycle — what was approved isn't what's on the file. Caught.
  • Icons missing — vegetarian mark, cruelty-free, recycle symbols dropped during a layout change. Caught.

A Claude skill built for the Hotline (same-day delivery) team that catches errors on packaging artwork before it goes to print. Upload a label, carton, or kit and it flags only the actual issues — FOP, BOP, compliance, copy quality, and manufacturer details.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude, upload the artwork as a PDF, and ask for a check — "review this," "is this ready?", "any issues?" all trigger the skill.
  3. Claude converts the file, reads both the visual and the extracted text, and runs the full check: FOP elements, net qty, MRP, USP, MADE IN INDIA, copy cleanliness, manufacturer block, batch fields, barcode, icons, Pantones.
  4. You get back a clean issue list, or "✅ All clear" if nothing's wrong.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The artwork file as a PDF — this format gives the cleanest read. JPG/PNG will work in a pinch but accuracy drops, especially on small text and the MRP/USP ₹ symbol.
  • The skill installed in your Claude skills directory
