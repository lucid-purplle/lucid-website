Skill

Alps Goodness Visual Universe

A Claude skill that gives Claude the full Soft Living visual universe for Alps Goodness — palette, soft natural daylight, lived-in space and props, and the core 'ritual already complete' rule (show the finished state, never the process). Use it to write or review prompts for any Alps visual so the result feels unmistakably on-brand.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and mention Alps Goodness or any Alps product (Ghee Moisturiser, Rosemary Water, Beetroot Powder, Ubtan, etc.) — the skill triggers automatically.
  3. Tell Claude what shot you're after — PDP image, banner, lifestyle still — and which product.
  4. Claude returns an image prompt grounded in the Soft Living world: window light, warm Indian interior, the woman already using the product, no process shots.
  5. Pair with the MUSE Model Casting skill when a model is in frame (the five Alps women live there, not here), and with NBP Whisperer for the final NBP / NB2 prompt mechanics.
  6. To review an existing image, paste it into Claude and ask if it's on-brand — the skill checks it against the universe rules.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • Higgsfield account with NBP / NB2 access for generating the images
  • Optionally pair with NBP Whisperer (prompt mechanics) and MUSE Model Casting (for the five Alps women)
