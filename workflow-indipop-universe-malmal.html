Skill

IndiPop Visual Universe

A Claude skill that gives Claude the IndiPop Malmal visual universe for Jahn. Superstar — the dusty Indian-craft palette ('every colour has mud in it'), the soft pastel studio backdrops, the dreamy atmospheric haze, and the product-by-product rules. Use it to write or review prompts so any IndiPop visual reads as one coherent campaign across studio, model, lifestyle, and still life shots.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and mention IndiPop, Malmal, Jahn. Superstar, or any Superstar product (Cushion Lipstick, Flush Blush Stick, BB Tint, Slick Kajal, etc.) — the skill triggers automatically.
  3. Tell Claude what you're making and which product. The skill chooses the right Malmal sub-register (Studio Malmal default, or the moodier atmospheric variant) before drafting.
  4. Claude returns an NBP / NB2 prompt that wears the Malmal package — palette qualified with the dusty / handloom register, the right atmospheric treatment, and the product's own colour as the chromatic anchor.
  5. To review an existing image, paste it into Claude and ask if it's Malmal-correct.
  6. Important: Krystal Bloom is a separate Jahn. universe (for the fragrance line). Do not mix the two.

What you'll need

  • Claude access (Pro, Team, or Max)
  • The skill installed in your Claude skills directory
  • Higgsfield account with NBP / NB2 access for generating the images
  • Optionally pair with the NBP Whisperer skill for the final prompt mechanics
