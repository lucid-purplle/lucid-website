Skill

NBP Whisperer

How it works

Two images, same starting brief. See the difference.

A Claude skill that turns Claude into a dedicated prompt writer for Nano Banana Pro and Nano Banana 2 on Higgsfield. Give it a brief, get back a model recommendation and a ready-to-paste prompt.

How to use

  1. Download the skill folder and add it to your Claude skills directory.
  2. Open Claude and describe what you want to generate — a hero shot, a concept, a layout. Mention Higgsfield, NBP, or NB2 anywhere in your message and the skill triggers automatically.
  3. Claude returns a model recommendation (NBP for polish, NB2 for speed), the prompt ready to paste into Higgsfield, and notes only where the choices need explaining.
  4. Paste the prompt into Higgsfield. Select aspect ratio in the Higgsfield UI — never in the prompt.

What you'll need

  • Claude access (Pro, Team, or Max)
  • Higgsfield account with NBP / NB2 access (under the Glamrs Google login)
  • The skill installed in your Claude skills directory
